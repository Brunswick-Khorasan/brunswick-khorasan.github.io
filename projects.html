Math Day Database

In 2024, I worked with Levi Heath to make a database for the problems used in one of UNL's math outreach events, Math Day. What this project does is to automate some of the more repetitive elements of generating a problem set, namely randomly selecting old problems and formatting them into a file.

This project was completed in building both the database and an interface for it, and should streamline some portions of our outreach work.

Private WoW / Azerothcore Server (modded 3.3.5a)

What started as a bit of a nostalgia project has become a lot of learning about & using SQL, database management, and debugging and maneuvering through a large project. I've run this for over a year, and it has been nice having something like this to tweak. Within this project was installing and updating an abandoned module that added a player race from a future expansion.

Math 203 Lecture & Chapter introduction Videos

As a part of my work preparing for an asynchronous section of Math 203 (UNL's math class for non-STEM majors), I was asked to rerecord some of the lecture videos, in order to make them match with updated content. This was really cool, and has sparked interest in making more videos to teach people various topics.

Later on, in order to try and replicate some of the connection of an in-person class, I made some quick videos to introduce the different chapters of the course, and to provide the connective tissue that I would normally include in my lessons.
